# Advanced Search for X (Twitter) 🔍

X (旧Twitter) の検索体験を劇的に向上させる、多機能なブラウザ拡張スクリプトです。
複雑な検索コマンドを覚える必要はありません。直感的な操作で高度な検索を行い、お気に入りやリストをフォルダで整理し、高度なミュート機能でタイムラインを快適にします。

![License](https://img.shields.io/badge/license-MIT-green.svg)

## ✨ 主な機能

このスクリプトは単なる検索補助ではありません。Xを「情報収集・分析ツール」として使いこなすための機能を網羅しています。

### 🚀 1. 強力な検索ビルダー (Search)
* **直感的な入力:** 「この言葉を含む」「この言葉を含まない」「最低100いいね以上」などの条件をフォームに入力するだけで、複雑な検索コマンドを自動生成します。
* **双方向同期:** 検索バーに直接入力したコマンドも解析され、このツールの画面に反映されます。検索コマンドの意味を解読するのにも役立ちます。
* **日付指定:** カレンダーでの期間指定はもちろん、「過去24時間」「過去1年」などのショートカットも完備。

### 📂 2. ドラッグ＆ドロップ整理 (Folders)
* **フォルダ管理:** 「保存した検索」「アカウント」「リスト」を、自分で作成したフォルダに入れて整理できます。
* **簡単操作:** アイテムをドラッグ＆ドロップするだけで、並び替えやフォルダ移動が可能です。

### ⭐ 3. 高機能ローカルブックマーク (Favorites)
* **★ボタンを追加:** ツイートのシェアボタン付近に「★」ボタンを追加します。クリックするだけで保存できます。
* **タグ管理:** ブックマークに色付きの「タグ」を付けて分類できます。階層構造もサポート。
* **相手に通知なし:** ブラウザ内に保存されるため、相手に通知がいくことはありません。また、画像や動画も一緒に記録されます。

### 🔇 4. 高度なミュート機能 (Mute)
* **強力なフィルタ:** 通常のミュートに加え、「正規表現（Regex）」を使った高度なフィルタリングが可能です。
* **柔軟な設定:** 「完全に消す（Hidden）」か、「折りたたんで表示（Collapsed）」かを選べます。
* **スコープ:** 本文だけでなく、ユーザー名やIDに対してもミュート判定を行えます。

### 🛡️ 5. プライバシーとデータ管理
* **ローカル保存:** 全てのデータ（履歴、ブックマーク、設定）はお使いのブラウザ内（Tampermonkey等のストレージ）に保存されます。外部サーバーに送信されることはありません。
* **シークレットモード:** 検索履歴を残さずに利用できるモードを搭載しています。
* **バックアップ:** 設定やデータはJSONファイルとして書き出し（エクスポート）・読み込み（インポート）が可能です。

## 📥 インストール方法

このツールを使用するには、ブラウザにユーザースクリプトマネージャー（拡張機能）が必要です。

1.  **マネージャーの導入:** 以下のいずれかの拡張機能をブラウザにインストールしてください。
    * [Violentmonkey](https://violentmonkey.github.io/)
    * [Tampermonkey](https://www.tampermonkey.net/)
2.  **スクリプトのインストール:**
    * [ここをクリックしてインストール](https://github.com/koyasi777/advanced-search-for-x-twitter/raw/main/x-advanced-search.user.js)
3.  **X (Twitter) を開く:**
    * ページを再読み込みすると、画面右上に「虫眼鏡アイコン 🔍」が出現します。

## 📖 使い方ガイド

### 基本操作
画面右上の **虫眼鏡アイコン** をクリックすると、操作パネル（モーダル）が開きます。
* **移動:** アイコンやパネルの上部をドラッグして好きな場所に配置できます。
* **リサイズ:** パネルの端をドラッグしてサイズを変更できます。
* **テーマ:** Xの表示設定（ライト/ダーク/ブラック）に合わせて自動で配色が切り替わります。

### タブの解説

| タブ | 機能説明 |
| :--- | :--- |
| **検索 (Search)** | 高度な検索条件を組み立てます。「検索実行」またはEnterキーで検索します。左下の「保存」で条件を保存できます。 |
| **履歴 (History)** | 過去の検索履歴を表示します。シークレットモード中は記録されません。 |
| **保存 (Saved)** | よく使う検索条件を保存・管理します。フォルダ分けが可能です。 |
| **お気に入り (Favorites)** | ★ボタンで保存したツイートを閲覧できます。タグによる絞り込みや検索が可能です。 |
| **ミュート (Mute)** | ミュートしたい単語や正規表現を登録・管理します。 |
| **リスト (Lists)** | 見ているリストを登録し、フォルダで整理して素早くアクセスできます。 |
| **アカウント (Accounts)** | 特定のユーザープロファイルを登録し、フォルダ管理できます。 |

## ⚙️ 設定とバックアップ

パネル右上の ⚙️ アイコンから設定画面を開けます。

* **UI言語:** 自動判定されますが、手動で日本語/英語などを切り替えられます。
* **タブ表示:** 使わないタブを非表示にしてメニューをすっきりさせることができます。
* **インポート/エクスポート:** 別のPCにデータを移行したり、バックアップを取るためにJSON形式でデータを保存できます。
